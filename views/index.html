A tiny MVC framework for Node

Given this route:

'/index': {
stuff: {
things: ['apples', 'oranges', 'bananas']
},
show: true
}

... this template:

{{if show}}

<ul>

{{for thing in stuff.things}}

<li>I like {{thing}}!</li>

{{endfor}}

</ul>

{{endif}}

<div>2 * 4 = {{ 2 * 4 }}</div>

... renders this:

{{if show}}
    {{for thing in stuff.things}}
  • I like {{thing}}!
    • {{endfor}}
{{endif}}
2 * 4 = {{ 2 * 4 }}
